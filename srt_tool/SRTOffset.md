# SRT Offset Tool

基于 tkinter 的 SRT 字幕时间偏移与文本编辑工具。

## 功能

- **打开文件**：点击工具栏"打开文件"按钮，或直接拖放 `.srt` 文件到窗口
- **字幕列表**：4 列显示（编号、开始时间、结束时间、文本内容），文本列占大部分宽度
- **多选**：Ctrl 选中不连续多行，Shift 选中连续一段
- **时间偏移**：选中行后"时间-" / "时间+"按钮亮起，点击弹出对话框输入毫秒数，对选中字幕的开始和结束时间统一加减，调整后立即存盘
- **文本编辑**：双击某行弹出文本编辑框，修改后确定即生效，同样立即存盘
- **撤销/重做**：支持时间偏移和文本编辑的撤销与重做，操作后立即存盘

## 依赖

- Python 3.7+
- tkinter（Python 自带）
- tkinterdnd2：拖放支持，`pip install tkinterdnd2`

## 使用

```bash
python SRTOffset.py
```
